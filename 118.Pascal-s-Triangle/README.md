###Problem
Given numRows, generate the first numRows of Pascal's triangle.  
`For example, given numRows = 5,`  
`Return`  
`[`  
&#160;&#160;&#160;&#160;&#160;    `[1],`  
&#160;&#160;&#160;&#160;   `[1,1],`  
&#160;&#160;&#160;  `[1,2,1],`  
&#160;&#160; `[1,3,3,1],`  
&#160;`[1,4,6,4,1]`  
`]`

###Ideas
打印pascal三角形。规则很简单，关系式也写来也不复杂，注意一些vector操作的细节即可。
